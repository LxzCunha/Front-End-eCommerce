Sobre o projeto

A MecanStore é um site acadêmico desenvolvido com HTML, CSS e JavaScript puro. O objetivo é simular uma loja de motores estacionários, exibindo produtos, permitindo adicionar itens ao carrinho e finalizar um pedido fictício.

HTML semântico

As páginas usam header, nav, main, section, article, table, form e footer.

CSS organizado

O layout utiliza cards, grid, flexbox, responsividade e estados visuais.

JavaScript puro

O projeto usa DOM, eventos, criação de elementos, validação e localStorage.
